# OpenJDK没有jstack等命令的解决办法

使用yum list --showduplicate | grep java-1.8 找出相关的软件版本

yum install 安装图中这两个软件即可![jstack.png](http://io.storyxc.com/storyxc/5b912b2afd7847419e5d2af77f733692.png)
